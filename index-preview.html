2 Métodos

2.1 Base de dados ICnet

O Sistema ICnet é um pacote de soluções de software que permite a gestão integrada de várias áreas das Instituições Conscienciocêntricas, UNICIN (2025):

  1. Voluntariado.
  2. Eventos.
  3. Gestão financeira (receitas e despesas) e contábil.
  4. Ecommerce.
  5. Dinâmicas parapsíquicas.
  6. Laboratórios.
  7. Hospedagem em Campus.
  8. Gescons.
  9. Gerenciamento de Ensino a Distância.
  10. Marketing Digital.

Através de uma conta com permissões admnistrativas de uma IC é possível acessar os dados cadastrados nos sistema.

Nas seções a seguir são apresentadas as telas contendo os dados registrado no sistema.

2.1.1 Voluntariado

Para acessar o módulo de gestão de Voluntariado é necessário logar com uma conta administrativa de usuário da IC.

A Figura 1 apresenta a tela de Voluntariado (para o voluntário Eduardo de Santana Medeiros) na IC Juriscons. Nela podemos verificar que “Eduardo de Santana Medeiros” está ativo no voluntariado (nenhuma data de saída), que está lotado na Sede da JURISCONS (em Foz do Iguaçu), está vinculado através do organograma da instituição em Colegiado Administrativo \ Relações Institucionais com função de Membro e iniciou formalmente o voluntariado em 11/06/2025.

A tela de “Voluntariado em Outras IC’s”, não apresentada mas acessível a partir desta tela, possui estrutura semelhante.

2.2 E-mail

O E-mail também uma rica fonte de informações, pois é habitual receber confirmações de compras ou tickets no e-mail.

2.3 Calendário

A agenda de compromissos é um ótimo recurso para consultar eventos anteriores (ver Figura 4).

2.4 WhatsApp

Alguns eventos podem ser pesquisados através do WhatsApp (Figura 5), como é o caso das atividades realizadas na OIC (que não usa o ICNet para registros de atendimentos).

2.5 Livros

As compras (ou recebimento de presentes) de livros podem ter registros no e-mail ou uma organização da biblioteca pessoal pode set implementada manualmente.

2.7 RStudio

O RStudio mantido por Posit Software, PBC (2025b) é uma IDE (Ambiente de Desenvolvimento Integrado) que permite a edição dos documentos e agrupa diversas funcionalidades para gerenciar o projeto. Ela é uma ferramenta de referência para a ciência de dados de código aberto.

Este documento foi redigido no RStudio.

2.8 Quarto

O Quarto mantido por Posit Software, PBC (2025a) é um sistema de código aberto para escrita e publicação técnica e científica. Ela é uma ferramenta de comandos de linhas para gerenciar e publicar projetos, integrada nas versões recentes do RStudio.
